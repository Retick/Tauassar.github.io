Описание:

Univero – это уникальная платформа по поступлению в зарубежные вузы, которая не имеет аналогов в Центральной Азии.

Здесь я прошел полный цикл создание дизайна (Изучение материалов -> ТЗ -> Скетч -> Вайрфрейм -> Прототип -> Дизайн). Для удобства верстальщиков я адаптировал всё под 5 экранов (Десктоп, Макбук, Планшет, Мобилка и iPhone5S). Работать приходилось в ускоренном темпе из-за сжатых сроков. Иногда пришлось редизайнить всё по двум основным причинам: заказчик вводил новую информацию которая координально все меняла; мне самому что-то не нравилось (честно говоря и в нынешнем состоянии еще есть над чем работать). Несколько скринов самого сайта ниже.

Из личного кабинета, который предназначен для самой подачи, было решено сделать приложение, поэтому несколько скринов ниже. 1 и 2 скрин с основного сайта, дальше личный кабинет.

Т.к. была вероятность создания других сайтов в такомже стиле, я наконец-то создал свой UI kit. Ниже скрины (не все)
